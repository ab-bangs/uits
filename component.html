Central Processing Unit (CPU)

Function: Executes instructions and processes all data.

Impact: Determines the overall speed and performance of the computer.

Details: Known as the “brain” of the computer. Measured in GHz. Examples include Intel and AMD processors.

Random Access Memory (RAM)

Function: Temporarily stores data while programs are running.

Impact: More RAM allows smoother multitasking.

Details: Volatile memory. Comes in sizes like 4GB, 8GB, 16GB and above.

Hard Drive / SSD

Function: Stores operating system, software and files.

Impact: SSD improves boot speed and system responsiveness.

Details: HDD (mechanical) and SSD (solid-state). Measured in GB or TB.

Motherboard

Function: Connects all hardware components together.

Impact: Determines compatibility of CPU, RAM and storage.

Details: Contains chipset, expansion slots, BIOS and connectors.

Power Supply Unit (PSU)

Function: Converts AC to DC power for the computer.

Impact: Stable power prevents hardware damage.

Details: Measured in watts (e.g., 450W, 600W).

Monitor

Function: Displays visual output.

Impact: Determines image clarity and resolution quality.

Details: LCD, LED, and 4K monitors available.

Keyboard

Function: Used for inputting text and commands.

Impact: Essential for productivity and programming.

Details: Mechanical and membrane types available.

Mouse

Function: Controls cursor movement.

Impact: Enables graphical interaction with software.

Details: Optical and wireless versions available.
